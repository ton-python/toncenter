---
title: "Detect Address"
openapi: "GET /api/v2/detectAddress"
---

## Parameters

| Parameter | Required | Description                    |
| --------- | -------- | ------------------------------ |
| `address` | Yes      | Address in any form.           |

## Response

Returns [`DetectAddress`](/api_v2/models#detectaddress).

| Field           | Type                         | Description                          |
| --------------- | ---------------------------- | ------------------------------------ |
| `raw_form`      | `str`                        | Raw address form.                    |
| `bounceable`    | `DetectAddressBase64Variant` | Bounceable base64/base64url forms.   |
| `non_bounceable`| `DetectAddressBase64Variant` | Non-bounceable base64/base64url forms.|
| `given_type`    | `str`                        | Type of the given address.           |
| `test_only`     | `bool`                       | Whether address is testnet-only.     |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = "EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.utils.detect_address(
            ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
