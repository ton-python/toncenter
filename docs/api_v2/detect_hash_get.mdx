---
title: "Detect Hash"
openapi: "GET /api/v2/detectHash"
---

## Parameters

| Parameter | Required | Description                    |
| --------- | -------- | ------------------------------ |
| `hash`    | Yes      | Hash in any form.              |

## Response

Returns [`DetectHash`](/api_v2/models#detecthash).

| Field    | Type  | Description              |
| -------- | ----- | ------------------------ |
| `b64`    | `str` | Base64 representation.   |
| `b64url` | `str` | Base64url representation.|
| `hex`    | `str` | Hex representation.      |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

HASH = "8085186e2c707c0de954e5456f065965988a01f7069bef6a6db32325af0e0d53"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.utils.detect_hash(
            HASH,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
