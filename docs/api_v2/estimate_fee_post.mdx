---
title: "Estimate Fee"
openapi: "POST /api/v2/estimateFee"
---

## Parameters

| Parameter       | Required | Description                                    |
| --------------- | -------- | ---------------------------------------------- |
| `address`       | Yes      | Destination address in any form.               |
| `body`          | Yes      | Base64-encoded message body BoC.               |
| `init_code`     | No       | Base64-encoded init code (for deploy).         |
| `init_data`     | No       | Base64-encoded init data (for deploy).         |
| `ignore_chksig` | No       | Skip signature check (default `True`).         |

## Response

Returns [`QueryFees`](/api_v2/models#queryfees).

| Field              | Type          | Description                      |
| ------------------ | ------------- | -------------------------------- |
| `source_fees`      | `Fees`        | Fees charged at source.          |
| `destination_fees` | `list[Fees]`  | Fees charged at destinations.    |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = "UQCDrgGaI6gWK-qlyw69xWZosurGxrpRgIgSkVsgahUtxZR0"
BODY = "te6ccgEBAQEAAgAAAA=="


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.send.estimate_fee(
            ADDRESS,
            BODY,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
