---
title: "Get Block Transactions Ext"
openapi: "GET /api/v2/getBlockTransactionsExt"
---

## Parameters

| Parameter    | Required | Description                                              |
| ------------ | -------- | -------------------------------------------------------- |
| `workchain`  | Yes      | Workchain ID.                                            |
| `shard`      | Yes      | Shard ID.                                                |
| `seqno`      | Yes      | Block sequence number.                                   |
| `root_hash`  | No       | Root hash for verification.                              |
| `file_hash`  | No       | File hash for verification.                              |
| `after_lt`   | No       | Return transactions after this logical time.             |
| `after_hash` | No       | Account hash (hex or base64) to read transactions after. |
| `count`      | No       | Maximum number of transactions to return.                |

## Response

Returns [`BlockTransactionsExt`](/api_v2/models#blocktransactionsext).

| Field          | Type                    | Description                                |
| -------------- | ----------------------- | ------------------------------------------ |
| `id`           | `TonBlockIdExt`         | Block identifier.                          |
| `req_count`    | `int`                   | Requested transaction count.               |
| `incomplete`   | `bool`                  | Whether more transactions are available.   |
| `transactions` | `list[TransactionExt]`  | Full transaction objects.                  |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

WORKCHAIN = -1
SHARD = "-9223372036854775808"
SEQNO = 57981181


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.transactions.get_block_transactions_ext(
            WORKCHAIN,
            SHARD,
            SEQNO,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
