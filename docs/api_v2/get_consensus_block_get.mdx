---
title: "Get Consensus Block"
openapi: "GET /api/v2/getConsensusBlock"
---

## Parameters

No parameters.

## Response

Returns [`ConsensusBlock`](/api_v2/models#consensusblock).

| Field             | Type  | Description                                 |
| ----------------- | ----- | ------------------------------------------- |
| `consensus_block` | `int` | Block number confirmed by validator consensus. |
| `timestamp`       | `int` | UNIX timestamp of the consensus block.      |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.blocks.get_consensus_block()
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
