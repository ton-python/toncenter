---
title: "Get Extended Address Information"
openapi: "GET /api/v2/getExtendedAddressInformation"
---

## Parameters

| Parameter | Required | Description                                   |
| --------- | -------- | --------------------------------------------- |
| `address` | Yes      | Account address in any form.                  |
| `seqno`   | No       | Masterchain block seqno for historical state.  |

## Response

Returns [`ExtendedAddressInformation`](/api_v2/models#extendedaddressinformation).

| Field                 | Type                         | Description                                 |
| --------------------- | ---------------------------- | ------------------------------------------- |
| `address`             | `AccountAddress`             | Account address object.                     |
| `balance`             | `str`                        | Account balance in nanotons.                |
| `extra_currencies`    | `list[ExtraCurrencyBalance]` | Extra currency balances.                    |
| `last_transaction_id` | `InternalTransactionId`      | Last transaction LT and hash.               |
| `block_id`            | `TonBlockIdExt`              | Block in which the state was retrieved.      |
| `sync_utime`          | `int`                        | Timestamp of last synchronization.           |
| `account_state`       | `AccountState`               | Parsed account state (discriminated union).  |
| `revision`            | `int`                        | Account revision number.                     |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = "EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.accounts.get_extended_address_information(
            ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
