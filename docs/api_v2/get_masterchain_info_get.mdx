---
title: "Get Masterchain Info"
openapi: "GET /api/v2/getMasterchainInfo"
---

## Parameters

No parameters.

## Response

Returns [`MasterchainInfo`](/api_v2/models#masterchaininfo).

| Field             | Type            | Description                            |
| ----------------- | --------------- | -------------------------------------- |
| `last`            | `TonBlockIdExt` | Last known masterchain block.          |
| `state_root_hash` | `str`           | State root hash.                       |
| `init`            | `TonBlockIdExt` | Initial masterchain block.             |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.blocks.get_masterchain_info()
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
