---
title: "Get Out Msg Queue Size"
openapi: "GET /api/v2/getOutMsgQueueSize"
---

## Parameters

No parameters.

## Response

Returns [`OutMsgQueueSizes`](/api_v2/models#outmsgqueuesizes).

| Field                    | Type                   | Description                          |
| ------------------------ | ---------------------- | ------------------------------------ |
| `shards`                 | `list[OutMsgQueueSize]`| Queue size per shard.                |
| `ext_msg_queue_size_limit`| `int`                 | External message queue size limit.   |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.blocks.get_out_msg_queue_size()
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
