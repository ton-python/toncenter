---
title: "Get Shard Block Proof"
openapi: "GET /api/v2/getShardBlockProof"
---

## Parameters

| Parameter    | Required | Description                                     |
| ------------ | -------- | ----------------------------------------------- |
| `workchain`  | Yes      | Workchain ID.                                   |
| `shard`      | Yes      | Shard ID.                                       |
| `seqno`      | Yes      | Block sequence number.                          |
| `from_seqno` | No       | Starting masterchain seqno for proof chain.     |

## Response

Returns [`ShardBlockProof`](/api_v2/models#shardblockproof).

| Field        | Type                  | Description                             |
| ------------ | --------------------- | --------------------------------------- |
| `from_block` | `TonBlockIdExt`       | Source block (aliased from `from`).     |
| `mc_id`      | `TonBlockIdExt`       | Masterchain block ID.                   |
| `links`      | `list[ShardBlockLink]`| Shard block links in the proof chain.   |
| `mc_proof`   | `list[BlockLinkBack]` | Masterchain proof links.               |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

WORKCHAIN = 0
SHARD = "-9223372036854775808"
SEQNO = 49560770


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.blocks.get_shard_block_proof(
            WORKCHAIN,
            SHARD,
            SEQNO,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
