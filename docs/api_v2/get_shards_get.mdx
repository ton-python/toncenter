---
title: "Get Shards"
openapi: "GET /api/v2/getShards"
---

## Parameters

| Parameter | Required | Description                            |
| --------- | -------- | -------------------------------------- |
| `seqno`   | Yes      | Masterchain block sequence number.     |

## Response

Returns [`Shards`](/api_v2/models#shards).

| Field    | Type                  | Description                   |
| -------- | --------------------- | ----------------------------- |
| `shards` | `list[TonBlockIdExt]` | Shard block descriptors.      |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

SEQNO = 57981181


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.blocks.get_shards(
            SEQNO,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
