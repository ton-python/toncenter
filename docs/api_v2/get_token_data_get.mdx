---
title: "Get Token Data"
openapi: "GET /api/v2/getTokenData"
---

## Parameters

| Parameter | Required | Description                            |
| --------- | -------- | -------------------------------------- |
| `address` | Yes      | Token contract address in any form.    |

## Response

Returns [`TokenData`](/api_v2/models#tokendata) — discriminated union of [`JettonMasterData`](/api_v2/models#jettonmasterdata), [`JettonWalletData`](/api_v2/models#jettonwalletdata), [`NftCollectionData`](/api_v2/models#nftcollectiondata), or [`NftItemData`](/api_v2/models#nftitemdata) based on `contract_type`.

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = "EQCxE6mUtQJKFnGfaROTKOt1lZbDiiX1kCixRv7Nw2Id_sDs"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.accounts.get_token_data(
            ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
