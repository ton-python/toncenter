---
title: "Get Transactions Std"
openapi: "GET /api/v2/getTransactionsStd"
---

## Parameters

| Parameter  | Required | Description                                    |
| ---------- | -------- | ---------------------------------------------- |
| `address`  | Yes      | Account address in any form.                   |
| `limit`    | No       | Maximum number of transactions (default 10).   |
| `lt`       | No       | Start from this logical time (pagination).     |
| `hash`     | No       | Start from this transaction hash (pagination). |
| `to_lt`    | No       | Return transactions up to this logical time.   |
| `archival` | No       | Use archival node for old transactions.        |

## Response

Returns [`TransactionsStd`](/api_v2/models#transactionsstd).

| Field                     | Type                    | Description                              |
| ------------------------- | ----------------------- | ---------------------------------------- |
| `transactions`            | `list[TransactionStd]`  | Transactions in standardized format.     |
| `previous_transaction_id` | `InternalTransactionId` | ID for paginating to older transactions. |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = "EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.transactions.get_transactions_std(
            ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
