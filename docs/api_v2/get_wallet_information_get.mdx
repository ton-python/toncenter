---
title: "Get Wallet Information"
openapi: "GET /api/v2/getWalletInformation"
---

## Parameters

| Parameter | Required | Description                                   |
| --------- | -------- | --------------------------------------------- |
| `address` | Yes      | Account address in any form.                  |
| `seqno`   | No       | Masterchain block seqno for historical state.  |

## Response

Returns [`WalletInformation`](/api_v2/models#walletinformation).

| Field                  | Type                    | Description                                          |
| ---------------------- | ----------------------- | ---------------------------------------------------- |
| `wallet`               | `bool`                  | Whether the account is a wallet contract.             |
| `balance`              | `str`                   | Account balance in nanotons.                          |
| `account_state`        | `AccountStateEnum`      | `"active"`, `"uninitialized"`, or `"frozen"`.         |
| `last_transaction_id`  | `InternalTransactionId` | Last transaction LT and hash.                         |
| `wallet_type`          | `str \| None`           | Wallet contract type name.                            |
| `seqno`                | `int \| None`           | Current wallet seqno.                                 |
| `wallet_id`            | `int \| None`           | Wallet ID (subwallet ID).                             |
| `is_signature_allowed` | `bool \| None`          | Whether external signatures are allowed.              |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = "UQCDrgGaI6gWK-qlyw69xWZosurGxrpRgIgSkVsgahUtxZR0"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.accounts.get_wallet_information(
            ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
