## Common

### `TonBlockIdExt`

Extended block identifier.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `workchain` | `int` | Workchain identifier. |
| `shard` | `str` | Shard identifier. |
| `seqno` | `int` | Block sequence number. |
| `root_hash` | `str` | Root hash of the block. |
| `file_hash` | `str` | File hash of the block. |

### `AccountAddress`

Account address wrapper.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `account_address` | `str` | Account address in any form. |

### `InternalTransactionId`

Internal transaction identifier.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `lt` | `str` | Logical time. |
| `hash` | `str` | Transaction hash. |

### `ExtraCurrencyBalance`

Extra currency balance entry.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | `int` | Extra currency identifier. |
| `amount` | `str` | Amount as a decimal string. |

### `AccountStateEnum`

Account state as a string enum.

| Value | Description |
| ----- | ----------- |
| `uninitialized` | Account has not been deployed. |
| `active` | Account is active with code and data. |
| `frozen` | Account is frozen. |

## Accounts

### `AddressInformation`

Full account state returned by `getAddressInformation`.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `balance` | `str` | Account balance in nanotons. |
| `extra_currencies` | `list[ExtraCurrencyBalance]` | Extra currency balances. |
| `last_transaction_id` | `InternalTransactionId` | Last transaction identifier. |
| `block_id` | `TonBlockIdExt` | Block in which the state was retrieved. |
| `code` | `str` | Base64-encoded smart contract code. |
| `data` | `str` | Base64-encoded smart contract data. |
| `frozen_hash` | `str` | Hash of the frozen state, empty if not frozen. |
| `sync_utime` | `int` | UNIX timestamp of the last sync. |
| `state` | `AccountStateEnum` | Account state. |
| `suspended` | `bool \| None` | Whether the account is suspended. |

### `ExtendedAddressInformation`

Extended account state with typed `account_state`, returned by `getExtendedAddressInformation`.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `address` | `AccountAddress` | Account address. |
| `balance` | `str` | Account balance in nanotons. |
| `extra_currencies` | `list[ExtraCurrencyBalance]` | Extra currency balances. |
| `last_transaction_id` | `InternalTransactionId` | Last transaction identifier. |
| `block_id` | `TonBlockIdExt` | Block in which the state was retrieved. |
| `sync_utime` | `int` | UNIX timestamp of the last sync. |
| `account_state` | `AccountState` | Discriminated union of account state variants. |
| `revision` | `int` | Account revision number. |

### `WalletInformation`

Wallet-specific account information returned by `getWalletInformation`.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `wallet` | `bool` | Whether the account is a known wallet contract. |
| `balance` | `str` | Account balance in nanotons. |
| `account_state` | `AccountStateEnum` | Account state. |
| `last_transaction_id` | `InternalTransactionId` | Last transaction identifier. |
| `wallet_type` | `str \| None` | Wallet contract type (e.g. `"wallet v4 r2"`). |
| `seqno` | `int \| None` | Current wallet sequence number. |
| `wallet_id` | `int \| None` | Wallet identifier. |
| `is_signature_allowed` | `bool \| None` | Whether signature-based authorization is allowed (wallet v5). |

### `AccountState`

Discriminated union of typed account states. The `@type` field (aliased as `type`) selects the variant:

| Variant | `@type` value | Key fields |
| ------- | ------------- | ---------- |
| `AccountStateRaw` | `raw.accountState` | `code`, `data`, `frozen_hash` |
| `AccountStateWalletV3` | `wallet.v3.accountState` | `wallet_id`, `seqno` |
| `AccountStateWalletV4` | `wallet.v4.accountState` | `wallet_id`, `seqno` |
| `AccountStateWalletHighloadV1` | `wallet.highload.v1.accountState` | `wallet_id`, `seqno` |
| `AccountStateWalletHighloadV2` | `wallet.highload.v2.accountState` | `wallet_id` |
| `AccountStateDns` | `dns.accountState` | `wallet_id` |
| `AccountStateRWallet` | `rwallet.accountState` | `wallet_id`, `seqno`, `unlocked_balance`, `config` |
| `AccountStatePChan` | `pchan.accountState` | `config`, `state`, `description` |
| `AccountStateUninited` | `uninited.accountState` | `frozen_hash` |

### `TokenData`

Discriminated union of token data variants. The `contract_type` field selects the variant.

#### `JettonMasterData`

| Field | Type | Description |
| ----- | ---- | ----------- |
| `address` | `str` | Jetton master contract address. |
| `contract_type` | `Literal["jetton_master"]` | Always `"jetton_master"`. |
| `total_supply` | `str` | Total supply as a decimal string. |
| `mintable` | `bool` | Whether new tokens can be minted. |
| `admin_address` | `str \| None` | Admin address, if set. |
| `jetton_content` | `TokenContent` | On-chain or off-chain metadata. |
| `jetton_wallet_code` | `str` | Base64-encoded jetton wallet code. |

#### `JettonWalletData`

| Field | Type | Description |
| ----- | ---- | ----------- |
| `address` | `str` | Jetton wallet contract address. |
| `contract_type` | `Literal["jetton_wallet"]` | Always `"jetton_wallet"`. |
| `balance` | `str` | Jetton balance as a decimal string. |
| `owner` | `str` | Owner wallet address. |
| `jetton` | `str` | Jetton master contract address. |
| `mintless_is_claimed` | `bool \| None` | Whether the mintless jetton has been claimed. |
| `jetton_wallet_code` | `str` | Base64-encoded jetton wallet code. |

#### `NftCollectionData`

| Field | Type | Description |
| ----- | ---- | ----------- |
| `address` | `str` | NFT collection contract address. |
| `contract_type` | `Literal["nft_collection"]` | Always `"nft_collection"`. |
| `next_item_index` | `str` | Index of the next item to be minted. |
| `owner_address` | `str \| None` | Collection owner address. |
| `collection_content` | `TokenContent` | On-chain or off-chain metadata. |

#### `NftItemData`

| Field | Type | Description |
| ----- | ---- | ----------- |
| `address` | `str` | NFT item contract address. |
| `contract_type` | `Literal["nft_item"]` | Always `"nft_item"`. |
| `init` | `bool` | Whether the NFT item is initialized. |
| `index` | `str` | Item index within the collection. |
| `collection_address` | `str \| None` | Parent collection address. |
| `owner_address` | `str \| None` | Current owner address. |
| `content` | `TokenContent \| DnsContent` | Token metadata or DNS content. |

#### `TokenContent`

| Field | Type | Description |
| ----- | ---- | ----------- |
| `type` | `str` | Content type: `"onchain"` or `"offchain"`. |
| `data` | `Any` | Content data (dict for on-chain, string URI for off-chain). |

#### `DnsContent`

| Field | Type | Description |
| ----- | ---- | ----------- |
| `domain` | `str` | DNS domain name. |
| `data` | `DnsRecordSet` | DNS records for this domain. |

#### `DnsRecordSet`

| Field | Type | Description |
| ----- | ---- | ----------- |
| `dns_next_resolver` | `DnsRecord \| None` | Next resolver DNS record. |
| `wallet` | `DnsRecord \| None` | Wallet DNS record. |
| `site` | `DnsRecord \| None` | Site DNS record. |
| `storage` | `DnsRecord \| None` | Storage DNS record. |

#### `DnsRecord`

Discriminated union of DNS record types. The `@type` field selects the variant:

| Variant | `@type` value | Key fields |
| ------- | ------------- | ---------- |
| `DnsRecordSmcAddress` | `dns_smc_address` | `smc_addr` (`SmcAddr`) |
| `DnsRecordNextResolver` | `dns_next_resolver` | `resolver` (`SmcAddr`) |
| `DnsRecordAdnlAddress` | `dns_adnl_address` | `adnl_addr` (`str`) |
| `DnsRecordStorageAddress` | `dns_storage_address` | `bag_id` (`str`) |

#### `SmcAddr`

Smart contract address with workchain and address.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `workchain_id` | `int` | Workchain identifier. |
| `address` | `str` | Account address in hex form. |

## Blocks

| Field | Type | Description |
| ----- | ---- | ----------- |
| `last` | `TonBlockIdExt` | Latest masterchain block identifier. |
| `state_root_hash` | `str` | State root hash. |
| `init` | `TonBlockIdExt` | Initial (genesis) block identifier. |

### `ConsensusBlock`

Latest consensus block information.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `consensus_block` | `int` | Consensus block sequence number. |
| `timestamp` | `int` | UNIX timestamp of the consensus block. |

### `Shards`

List of shard block identifiers for a given masterchain seqno.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `shards` | `list[TonBlockIdExt]` | Shard block identifiers. |

### `BlockHeader`

Block header information.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | `TonBlockIdExt` | Extended identifier of the block. |
| `global_id` | `int` | Global network identifier. |
| `version` | `int` | Block format version. |
| `after_merge` | `bool` | True if block was created after a merge. |
| `after_split` | `bool` | True if block was created after a split. |
| `before_split` | `bool` | True if block was created before a split. |
| `want_merge` | `bool` | Indicates if validators wanted a merge. |
| `want_split` | `bool` | Indicates if validators wanted a split. |
| `validator_list_hash_short` | `int` | Short hash of validator list. |
| `catchain_seqno` | `int` | Catchain sequence number. |
| `min_ref_mc_seqno` | `int` | Minimum referenced masterchain seqno. |
| `is_key_block` | `bool` | True if this block is a key block. |
| `prev_key_block_seqno` | `int` | Previous key block sequence number. |
| `start_lt` | `str` | Starting logical time. |
| `end_lt` | `str` | Ending logical time. |
| `gen_utime` | `int` | Block generation UNIX timestamp. |
| `prev_blocks` | `list[TonBlockIdExt]` | List of previous block identifiers. |

### `BlockSignatures`

Block validator signatures.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | `TonBlockIdExt` | Block identifier. |

| Field | Type | Description |

| `node_id_short` | `str` | Short identifier of the validator node. |

### `ShardBlockProof`

Proof linking a shard block to a masterchain block.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `from_block` | `TonBlockIdExt` | Source shard block identifier (JSON key: `from`). |
| `mc_id` | `TonBlockIdExt` | Masterchain block identifier. |
| `links` | `list[ShardBlockLink]` | Chain of shard block links. |

### `ShardBlockLink`

| Field | Type | Description |
| ----- | ---- | ----------- |

| `proof` | `str` | Base64-encoded proof data. |

### `BlockLinkBack`

Backward link in a masterchain proof chain.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `to_key_block` | `bool` | Whether this links to a key block. |
| `from_block` | `TonBlockIdExt` | Source block identifier (JSON key: `from`). |

| `dest_proof` | `str` | Base64-encoded destination proof. |
| `proof` | `str` | Base64-encoded proof data. |
| `state_proof` | `str` | Base64-encoded state proof. |

Outgoing message queue sizes across all shards.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `shards` | `list[OutMsgQueueSize]` | List of outgoing message queue sizes per shard. |
| `ext_msg_queue_size_limit` | `int` | Limit for the external message queue size. |

### `OutMsgQueueSize`

Outgoing message queue size for a single shard.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | `TonBlockIdExt` | Block identifier for the shard. |
| `size` | `int` | Queue size for the shard. |

## Transactions

### `Transaction`

Extended transaction with string addresses and decoded messages. Returned by `getTransactions`.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `address` | `AccountAddress` | Account address (structured). |
| `account` | `str` | Account address as a string. |
| `utime` | `int` | UNIX timestamp of the transaction. |
| `data` | `str` | Base64-encoded raw transaction data. |
| `transaction_id` | `InternalTransactionId` | Transaction identifier. |
| `fee` | `str` | Total fee in nanotons. |
| `storage_fee` | `str` | Storage fee in nanotons. |
| `other_fee` | `str` | Other fees in nanotons. |
| `in_msg` | `Message \| None` | Incoming message, if present. |
| `out_msgs` | `list[Message]` | Outgoing messages. |

### `TransactionStd`

Standard transaction with structured `AccountAddress` source/destination in messages. Used in `TransactionsStd`.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `address` | `AccountAddress` | Account address. |
| `utime` | `int` | UNIX timestamp of the transaction. |
| `data` | `str` | Base64-encoded raw transaction data. |
| `transaction_id` | `InternalTransactionId` | Transaction identifier. |
| `fee` | `str` | Total fee in nanotons. |
| `storage_fee` | `str` | Storage fee in nanotons. |
| `other_fee` | `str` | Other fees in nanotons. |
| `in_msg` | `MessageStd \| None` | Incoming message, if present. |
| `out_msgs` | `list[MessageStd]` | Outgoing messages. |

### `TransactionExt`

Extended transaction used in `BlockTransactionsExt`. Same structure as `TransactionStd` with
an additional `account` field.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `address` | `AccountAddress` | Account address (structured). |
| `account` | `str` | Account address as a string. |
| `utime` | `int` | UNIX timestamp of the transaction. |
| `data` | `str` | Base64-encoded raw transaction data. |
| `transaction_id` | `InternalTransactionId` | Transaction identifier. |
| `fee` | `str` | Total fee in nanotons. |
| `storage_fee` | `str` | Storage fee in nanotons. |
| `other_fee` | `str` | Other fees in nanotons. |
| `in_msg` | `MessageStd \| None` | Incoming message, if present. |
| `out_msgs` | `list[MessageStd]` | Outgoing messages. |

### `TransactionsStd`

Paginated list of standard transactions returned by `getTransactionsStd`.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `transactions` | `list[TransactionStd]` | List of transactions. |
| `previous_transaction_id` | `InternalTransactionId` | Cursor for fetching the next page. |

### `BlockTransactions`

Block transactions with short identifiers.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | `TonBlockIdExt` | Identifier of the block containing the transactions. |
| `req_count` | `int` | Number of requested transactions. |
| `incomplete` | `bool` | Indicates if the transaction list is incomplete. |
| `transactions` | `list[ShortTxId]` | List of short transaction identifiers. |

### `ShortTxId`

Short transaction identifier within a block.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `mode` | `int` | Transaction mode flags. |
| `account` | `str` | Account address. |
| `lt` | `str` | Logical time of the transaction. |
| `hash` | `str` | Base64 hash of the transaction. |

### `BlockTransactionsExt`

Block transactions with full transaction data.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | `TonBlockIdExt` | Identifier of the block containing the transactions. |
| `req_count` | `int` | Number of requested transactions. |
| `incomplete` | `bool` | Indicates if the transaction list is incomplete. |
| `transactions` | `list[TransactionExt]` | List of full transactions. |

### `Message`

Extended message with string addresses and optional decoded content.

| Field | Type | Description |
| ----- | ---- | ----------- |
| `hash` | `str` | Message hash. |
| `source` | `str` | Source address as a string. |
| `destination` | `str` | Destination address as a string. |
| `value` | `str` | Transferred value in nanotons. |
| `extra_currencies` | `list[ExtraCurrencyBalance]` | Extra currency transfers. |
| `fwd_fee` | `str` | Forward fee in nanotons. |
| `ihr_fee` | `str` | IHR fee in nanotons. |
| `created_lt` | `str` | Logical time when the message was created. |
| `body_hash` | `str` | Hash of the message body. |
| `msg_data` | `MsgData` | Message payload. |
| `message` | `str \| None` | Decoded message content, if available. |
| `message_decode_error` | `str \| None` | Decode error description, if decoding failed. |

### `MessageStd`

Standard message with structured `AccountAddress` source/destination.

| Field | Type | Description |
| ----- | ---- | ----------- |

| `source` | `AccountAddress` | Source address. |
| `destination` | `AccountAddress` | Destination address. |
| `value` | `str` | Transferred value in nanotons. |
| `extra_currencies` | `list[ExtraCurrencyBalance]` | Extra currency transfers. |
| `fwd_fee` | `str` | Forward fee in nanotons. |
| `ihr_fee` | `str` | IHR fee in nanotons. |
| `created_lt` | `str` | Logical time when the message was created. |
| `body_hash` | `str` | Hash of the message body. |
| `msg_data` | `MsgData` | Message payload. |

### `MsgData`

| Variant | `@type` value | Key fields |
| ------- | ------------- | ---------- |
| `MsgDataRaw` | `msg.dataRaw` | `body`, `init_state` (both `str \| None`) |
| `MsgDataText` | `msg.dataText` | `text` (`str \| None`) |
| `MsgDataDecryptedText` | `msg.dataDecryptedText` | `text` (`str \| None`) |
| `MsgDataEncryptedText` | `msg.dataEncryptedText` | `text` (`str \| None`) |

| Field | Type | Description |
| ----- | ---- | ----------- |

| `block_id` | `TonBlockIdExt` | Block in which the method was executed. |
| `last_transaction_id` | `InternalTransactionId` | Last transaction identifier at execution time. |

### `RunGetMethodStdResult`

| Field | Type | Description |
| ----- | ---- | ----------- |
