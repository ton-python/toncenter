---
title: "Run Get Method"
openapi: "POST /api/v2/runGetMethod"
---

## Parameters

| Parameter | Required | Description                                       |
| --------- | -------- | ------------------------------------------------- |
| `address` | Yes      | Smart contract address in any form.                |
| `method`  | Yes      | Method name or numeric ID.                         |
| `stack`   | No       | Input stack entries (legacy 2-tuple format).        |
| `seqno`   | No       | Masterchain block seqno.                            |

## Response

Returns [`RunGetMethodResult`](/api_v2/models#rungetmethodresult).

| Field                 | Type                    | Description                         |
| --------------------- | ----------------------- | ----------------------------------- |
| `gas_used`            | `int`                   | Gas consumed by execution.          |
| `stack`               | `list[list[Any]]`       | Output stack (legacy format).       |
| `exit_code`           | `int`                   | TVM exit code.                      |
| `block_id`            | `TonBlockIdExt`         | Block used for execution context.   |
| `last_transaction_id` | `InternalTransactionId` | Last transaction of the contract.   |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = "EQC3dNlesgVD8YbAazcauIrXBPfiVhMMr5YYk2in0Mtsz0Bz"
METHOD = "dnsresolve"
STACK = [
    ["tvm.Slice", "te6ccgEBAQEABwAACm5lc3MA"],
    ["num", "0x19f02441ee588fdb26ee24b2568dd035c3c9206e11ab979be62e55558a1d17ff"],
]


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.runmethod.run_get_method(
            ADDRESS,
            METHOD,
            STACK,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
