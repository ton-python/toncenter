---
title: "Try Locate Source Tx"
openapi: "GET /api/v2/tryLocateSourceTx"
---

## Parameters

| Parameter     | Required | Description                                |
| ------------- | -------- | ------------------------------------------ |
| `source`      | Yes      | Source address in any form.                |
| `destination` | Yes      | Destination address in any form.           |
| `created_lt`  | Yes      | Creation logical time of the message.      |

## Response

Returns [`Transaction`](/api_v2/models#transaction). See [Get Transactions](/api_v2/get_transactions_get) for field details.

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

SOURCE = "EQAe8YtZ_XIHYK2An1R9xXIKedq3ghag2nX3PFScxcZQ_xHV"
DESTINATION = "EQBAjaOyi2wGWlk-EDkSabqqnF-MrrwMadnwqrurKpkla9nE"
CREATED_LT = "68403494000002"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.transactions.try_locate_source_tx(
            SOURCE,
            DESTINATION,
            CREATED_LT,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
