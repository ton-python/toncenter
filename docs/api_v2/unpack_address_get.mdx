---
title: "Unpack Address"
openapi: "GET /api/v2/unpackAddress"
---

## Parameters

| Parameter | Required | Description                        |
| --------- | -------- | ---------------------------------- |
| `address` | Yes      | User-friendly base64 address.      |

## Response

Returns `str`. Raw address string.

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = "EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v2.utils.unpack_address(
            ADDRESS,
        )
        print(result)


if __name__ == "__main__":
    asyncio.run(main())
```
