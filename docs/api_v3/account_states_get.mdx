---
title: "Get Account States"
openapi: "GET /api/v3/accountStates"
---

## Parameters

| Parameter     | Required | Description                                        |
| ------------- | -------- | -------------------------------------------------- |
| `address`     | Yes      | List of addresses in any form. Max 1000.           |
| `include_boc` | No       | Include code and data BOCs (default `True`).       |

## Response

Returns [`AccountStatesResponse`](/api_v3/models#accountstatesresponse).

| Field          | Type                         | Description                    |
| -------------- | ---------------------------- | ------------------------------ |
| `accounts`     | `list[AccountStateFull] \| None` | Account state objects.     |
| `address_book` | `AddressBook \| None`        | Address book entries.          |
| `metadata`     | `Metadata \| None`           | Address metadata.              |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = ["EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"]


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.accounts.get_account_states(
            address=ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
