---
title: "Get Address Book"
openapi: "GET /api/v3/addressBook"
---

## Parameters

| Parameter | Required | Description                                        |
| --------- | -------- | -------------------------------------------------- |
| `address` | Yes      | List of addresses in any form. Max 1024.           |

## Response

Returns [`AddressBook`](/api_v3/models#addressbook) — a mapping of raw address to [`AddressBookRow`](/api_v3/models#addressbookrow) with fields `user_friendly`, `domain`, and `interfaces`.

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = ["EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"]


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.accounts.get_address_book(
            address=ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
