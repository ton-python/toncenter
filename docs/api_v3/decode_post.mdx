---
title: "Post Decode"
openapi: "POST /api/v3/decode"
---

## Parameters

| Parameter | Required | Description                                                                |
| --------- | -------- | -------------------------------------------------------------------------- |
| `body`    | Yes      | `DecodeRequest` with `opcodes` and/or `bodies` fields. Use POST for long values. |

## Response

Returns [`DecodeResponse`](/api_v3/models#decoderesponse).

| Field     | Type                              | Description                    |
| --------- | --------------------------------- | ------------------------------ |
| `opcodes` | `list[str] \| None`               | Decoded opcode names.          |
| `bodies`  | `list[dict[str, Any]] \| None`    | Decoded message bodies.        |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network
from toncenter.rest.v3.models import DecodeRequest

OPCODES = ["0x0f8a7ea5"]


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.utils.post_decode(
            body=DecodeRequest(opcodes=OPCODES),
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
