---
title: "Get Dns Records"
openapi: "GET /api/v3/dns/records"
---

## Parameters

| Parameter | Required | Description                                                              |
| --------- | -------- | ------------------------------------------------------------------------ |
| `wallet`  | No       | Wallet address in any form. Returns DNS records containing this address. |
| `domain`  | No       | Exact domain name to search for.                                         |
| `limit`   | No       | Max rows to return (default 100).                                        |
| `offset`  | No       | Skip first N rows (default 0).                                           |

## Response

Returns [`DNSRecordsResponse`](/api_v3/models#dnsrecordsresponse).

| Field          | Type                        | Description                    |
| -------------- | --------------------------- | ------------------------------ |
| `records`      | `list[DNSRecord] \| None`   | DNS record objects.            |
| `address_book` | `AddressBook \| None`       | Address book entries.          |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

DOMAIN = "ness.ton"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.dns.get_dns_records(
            domain=DOMAIN,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
