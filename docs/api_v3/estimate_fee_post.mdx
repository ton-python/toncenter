---
title: "Estimate Fee"
openapi: "POST /api/v3/estimateFee"
---

## Parameters

| Parameter | Required | Description                                                                           |
| --------- | -------- | ------------------------------------------------------------------------------------- |
| `body`    | Yes      | `V2EstimateFeeRequest` with `address`, `body`, `init_code`, `init_data`, `ignore_chksig`. |

## Response

Returns [`V2EstimateFeeResult`](/api_v3/models#v2estimatefeeresult).

| Field              | Type                              | Description                    |
| ------------------ | --------------------------------- | ------------------------------ |
| `source_fees`      | `V2EstimatedFee \| None`          | Fees charged at source.        |
| `destination_fees` | `list[V2EstimatedFee] \| None`    | Fees charged at destinations.  |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network
from toncenter.rest.v3.models import V2EstimateFeeRequest

ADDRESS = "UQCDrgGaI6gWK-qlyw69xWZosurGxrpRgIgSkVsgahUtxZR0"
BODY = "te6ccgEBAQEAAgAAAA=="


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.api_v2.estimate_fee(
            body=V2EstimateFeeRequest(address=ADDRESS, body=BODY),
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
