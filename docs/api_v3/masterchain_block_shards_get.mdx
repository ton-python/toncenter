---
title: "Get Masterchain Block Shards"
openapi: "GET /api/v3/masterchainBlockShards"
---

## Parameters

| Parameter | Required | Description                                 |
| --------- | -------- | ------------------------------------------- |
| `seqno`   | Yes      | Masterchain block seqno.                    |
| `limit`   | No       | Max rows to return (default 10).            |
| `offset`  | No       | Skip first N rows (default 0).              |

## Response

Returns [`BlocksResponse`](/api_v3/models#blocksresponse).

| Field    | Type                     | Description                                               |
| -------- | ------------------------ | --------------------------------------------------------- |
| `blocks` | `list[Block] \| None`    | Workchain blocks after the previous masterchain block.    |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

SEQNO = 57981181


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.blockchain.get_masterchain_block_shards(
            seqno=SEQNO,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
