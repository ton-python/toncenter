---
title: "Get Masterchain Info"
openapi: "GET /api/v3/masterchainInfo"
---

## Parameters

No parameters.

## Response

Returns [`MasterchainInfo`](/api_v3/models#masterchaininfo).

| Field   | Type             | Description                    |
| ------- | ---------------- | ------------------------------ |
| `first` | `Block \| None`  | First indexed block.           |
| `last`  | `Block \| None`  | Last indexed block.            |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.blockchain.get_masterchain_info()
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
