---
title: "Get Metadata"
openapi: "GET /api/v3/metadata"
---

## Parameters

| Parameter | Required | Description                                        |
| --------- | -------- | -------------------------------------------------- |
| `address` | Yes      | List of addresses in any form. Max 1024.           |

## Response

Returns [`Metadata`](/api_v3/models#metadata) — a mapping of raw address to [`AddressMetadata`](/api_v3/models#addressmetadata) with fields `is_indexed` and `token_info`.

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = ["EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"]


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.accounts.get_metadata(
            address=ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
