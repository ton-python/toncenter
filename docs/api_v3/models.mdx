## `FinalityState`

Finality level of a transaction or action. Lifecycle: `pending` -> `confirmed` -> `finalized`.
Unknown values returned by the API are preserved as plain strings.

| Value | Description |
| ----- | ----------- |
| `pending` | Transaction is in the mempool, not yet included in a block. |
| `confirmed` | Transaction is included in a block but not yet finalized. |
| `finalized` | Transaction is included in a finalized block. |

## `AddressBook`

`RootModel[dict[str, AddressBookRow]]` mapping raw account addresses to their human-readable representations.

## `AddressBookRow`

| Field | Type | Description |
