---
title: "Get Nft Sales"
openapi: "GET /api/v3/nft/sales"
---

## Parameters

| Parameter | Required | Description                                                    |
| --------- | -------- | -------------------------------------------------------------- |
| `address` | Yes      | List of sale/auction contract addresses in any form. Max 1000. |

## Response

Returns [`NFTSalesResponse`](/api_v3/models#nftsalesresponse).

| Field          | Type                        | Description                    |
| -------------- | --------------------------- | ------------------------------ |
| `nft_sales`    | `list[NFTSale] \| None`     | NFT sale/auction objects.      |
| `address_book` | `AddressBook \| None`       | Address book entries.          |
| `metadata`     | `Metadata \| None`          | Address metadata.              |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = ["EQAMrsze7MaG_7P2ENd1eeT-S2VttJ1myT9sX5f-F1gY7xGx"]


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.nfts.get_nft_sales(
            address=ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
