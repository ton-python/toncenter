---
title: "Get Nft Transfers"
openapi: "GET /api/v3/nft/transfers"
---

## Parameters

| Parameter            | Required | Description                                                   |
| -------------------- | -------- | ------------------------------------------------------------- |
| `owner_address`      | No       | List of NFT owner addresses in any form. Max 1000.            |
| `item_address`       | No       | List of NFT item addresses in any form. Max 1000.             |
| `collection_address` | No       | Collection address in any form.                               |
| `direction`          | No       | Direction of transfer.                                        |
| `start_utime`        | No       | Filter: after given timestamp.                                |
| `end_utime`          | No       | Filter: before given timestamp.                               |
| `start_lt`           | No       | Filter: `lt >= start_lt`.                                      |
| `end_lt`             | No       | Filter: `lt <= end_lt`.                                        |
| `limit`              | No       | Max rows to return (default 10).                              |
| `offset`             | No       | Skip first N rows (default 0).                                |
| `sort`               | No       | Sort by lt (default `"desc"`).                                 |

## Response

Returns [`NFTTransfersResponse`](/api_v3/models#nfttransfersresponse).

| Field           | Type                            | Description                    |
| --------------- | ------------------------------- | ------------------------------ |
| `nft_transfers` | `list[NFTTransfer] \| None`     | NFT transfer records.          |
| `address_book`  | `AddressBook \| None`           | Address book entries.          |
| `metadata`      | `Metadata \| None`              | Address metadata.              |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

COLLECTION_ADDRESS = "EQC3dNlesgVD8YbAazcauIrXBPfiVhMMr5YYk2in0Mtsz0Bz"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.nfts.get_nft_transfers(
            collection_address=COLLECTION_ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
