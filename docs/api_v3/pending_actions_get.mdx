---
title: "Get Pending Actions"
openapi: "GET /api/v3/pendingActions"
---

## Parameters

| Parameter                | Required | Description                                               |
| ------------------------ | -------- | --------------------------------------------------------- |
| `account`                | No       | Account address in any form.                              |
| `ext_msg_hash`           | No       | List of trace external hashes.                            |
| `supported_action_types` | No       | Supported action types.                                   |
| `include_transactions`   | No       | Include `transactions_full` array (default `False`).      |

## Response

Returns [`ActionsResponse`](/api_v3/models#actionsresponse).

| Field          | Type                       | Description                    |
| -------------- | -------------------------- | ------------------------------ |
| `actions`      | `list[Action] \| None`     | Pending action objects.        |
| `address_book` | `AddressBook \| None`      | Address book entries.          |
| `metadata`     | `Metadata \| None`         | Address metadata.              |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ACCOUNT = "EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.actions.get_pending_actions(
            account=ACCOUNT,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
