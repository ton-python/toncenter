---
title: "Get Top Accounts By Balance"
openapi: "GET /api/v3/topAccountsByBalance"
---

## Parameters

| Parameter | Required | Description                                 |
| --------- | -------- | ------------------------------------------- |
| `limit`   | No       | Max rows to return (default 10).            |
| `offset`  | No       | Skip first N rows (default 0).              |

## Response

Returns `list[`[`AccountBalance`](/api_v3/models#accountbalance)`]`. Each entry contains:

| Field     | Type             | Description                    |
| --------- | ---------------- | ------------------------------ |
| `account` | `str \| None`    | Account address.               |
| `balance` | `str \| None`    | Account balance in nanotons.   |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.stats.get_top_accounts_by_balance()
        for account in result:
            print(account.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
