---
title: "Get Vesting Contracts"
openapi: "GET /api/v3/vesting"
---

## Parameters

| Parameter          | Required | Description                                                    |
| ------------------ | -------- | -------------------------------------------------------------- |
| `contract_address` | No       | List of vesting contract addresses in any form. Max 1000.      |
| `wallet_address`   | No       | List of wallet addresses (owner/sender) in any form. Max 1000. |
| `check_whitelist`  | No       | Check if wallet address is in whitelist (default `False`).     |
| `limit`            | No       | Max rows to return (default 10).                               |
| `offset`           | No       | Skip first N rows (default 0).                                 |

## Response

Returns [`VestingContractsResponse`](/api_v3/models#vestingcontractsresponse).

| Field               | Type                               | Description                    |
| ------------------- | ---------------------------------- | ------------------------------ |
| `vesting_contracts` | `list[IndexVestingInfo] \| None`   | Vesting contract objects.      |
| `address_book`      | `AddressBook \| None`              | Address book entries.          |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

WALLET_ADDRESS = ["UQCDrgGaI6gWK-qlyw69xWZosurGxrpRgIgSkVsgahUtxZR0"]


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.vesting.get_vesting_contracts(
            wallet_address=WALLET_ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
