---
title: "Get Wallet States"
openapi: "GET /api/v3/walletStates"
---

## Parameters

| Parameter | Required | Description                                        |
| --------- | -------- | -------------------------------------------------- |
| `address` | Yes      | List of addresses in any form. Max 1000.           |

## Response

Returns [`WalletStatesResponse`](/api_v3/models#walletstatesresponse).

| Field          | Type                        | Description                    |
| -------------- | --------------------------- | ------------------------------ |
| `wallets`      | `list[WalletState] \| None` | Wallet state objects.          |
| `address_book` | `AddressBook \| None`       | Address book entries.          |
| `metadata`     | `Metadata \| None`          | Address metadata.              |

## SDK Usage

```python
import asyncio

from toncenter.rest import ToncenterRestClient
from toncenter.types import Network

ADDRESS = ["EQDtFpEwcFAEcRe5mLVh2N6C0x-_hJEM7W61_JLnSF74p4q2"]


async def main() -> None:
    async with ToncenterRestClient(network=Network.MAINNET) as client:
        result = await client.v3.accounts.get_wallet_states(
            address=ADDRESS,
        )
        print(result.model_dump_json(indent=4))


if __name__ == "__main__":
    asyncio.run(main())
```
